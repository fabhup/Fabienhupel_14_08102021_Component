# DataTable Component <br/>

### **<span style={{color: 'grey',fontSize:'1.2em'}}> &nbsp;&nbsp;`<DataTable data={data} columns={columns} ...otherOptionalProps />`</span>**<br/><br/>

### **This component can be used to display data in a Datatable React Component.**

### **It can be customized with several optional props. Several parameters component can be used to display data informations in a Datatable React Component.**

<br />

### **[ - Required Props](#required-props)**

### **[ - Optional Props](#optional-props)**

## Required Props

<br />

-   ### _**data**_ :

    -   This Prop is **Required**
    -   **Format** : Array of objects (each object represents a row of data)
    -   **row.keys** : keys of rows objects _**must be the same as columns.key**_
    -   **row.values** : format string, number, or date (dates as string)
    -   _**Example**_ :

            [
                {
                    id: 1,
                    firstName: 'John',
                    lastName: 'Parks',
                    dateOfBirth: '1980-10-01',
                    startDate: '2015-09-01',
                    department: 'Human Resources',
                    street: '10 Downing Street',
                    city: 'London',
                    state: 'UK',
                    zipCode: 'SW1A2AB',
                },
                {
                    id: 2,
                    firstName: 'Peter',
                    lastName: 'Edwards',
                    dateOfBirth: '1958-03-15',
                    startDate: '2007-11-29',
                    department: 'IT',
                    street: '5 Helmet Row',
                    city: 'London',
                    state: 'UK',
                    zipCode: 'EC1V3QJ',
                },
                ...
            ]

        <br />
        <br />

-   ### _**columns**_ :

    -   This Prop is **Required**
    -   **Format** : Array of objects (columns=[column1, column2, ...]).
        Each column item must be an object with specific keys
    -   **column.Keys ** : used to specified specific parameters on each column (several required parameters and optionals parameters)

        > **Required keys** for each column object :
        >
        > -   **title**: title of the column to display on dataTableContentHeader
        > -   **key**: key of the column (must be the same as "data.row" keys)
        > -   **format**: format of values in data.row[column.key] (possible values : 'string', 'number' or 'date')
        >
        > **Optional keys** for each column object :
        >
        > -   **splitContent**: if true, content will be split if necessary on several rows if greater than width of column (boolean, default to false)
        > -   **splitContentRows**: max number of rows for split rows if splitContent is set to true. (number, default to 1)
        > -   **widthRatio**: parameter to change maxWidth of a column (ex : 2 or 3 for a large column & 0.5 for a short column) (number, default to 1) <br /> <br />

    -   _**Example**_ :

            [
                {
                    title: 'First Name',  // required : the title of column to display on dataTableContentHeader
                    key: 'firstName', // required : the key of column (must be the same as "data" keys)
                    format: 'string', // required : used to define format of column values to apply sort data depending on format
                    splitContent: true, // optional (default to false) : used to define if content can be split on several rows
                    splitContentRows: 2, // optional (default to 1): used to define the max number of rows for split rows
                    widthRatio: 1, // optional (default to 1): used (default to 1) to change the maxWidth of a column (ex : 2 or 3 for large columns 0.5 for short columns)
                },

                {
                    title: 'Last Name',
                    key: 'lastName',
                    format: 'string',
                },
                ...
            ]

## Optional Props

<br />

-   ### _**striped**_ :

    -   This Prop is **Optional**
    -   **Default Value** : **true**
    -   **Format** : Boolean
    -   **Description** : parameter to active or disable striped rows (if true, striped rows is activated)

    <br />

-   ### _**customColors**_ :

    -   This Prop is **Optional**
    -   **Format** : Object of colors (customColors={primaryColor: 'red', primaryColorBackground:'#ffffff'...}).
    -   **Description** : parameter to apply custom colors on dataTable elements
    -   **Default Value** : defaultColors
    -   **Possible Keys** :
        > -   **primaryColor** (optional : defaultColors.primaryColor if not defined)
        > -   **primaryColorBackground** (optional : defaultColors.primaryColorBackground if not defined)
        > -   **secondaryColor** (optional : defaultColors.secondaryColor if not defined)
        > -   **secondaryColorBackground** (optional : defaultColors.secondaryColorBackground if not defined)
        > -   **stripedColor** (optional : defaultColors.stripedColor if not defined)
        > -   **stripedColorBackground** (optional : defaultColors.stripedColorBackground if not defined)<br /> <br />
    -   _**Example**_ :

            {
                primaryColor: '#545454',
                primaryColorBackground: 'white',
                secondaryColor: '#73797e',
                secondaryColorBackground: '#ffffff',
                stripedColor: '#5a5a5a',
                stripedColorBackground: '#f2f4f6',
            }

        <br />

-   ### _**primaryColor**_ :
